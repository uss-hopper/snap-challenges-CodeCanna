Github Branching

Branches

  • Master
  • Develop
  • Feature
  • Release
  • Hotfix

Branch Discriptions:

Master

The master branch is where all of the main changes get pushed to for a release. This is where the main release of a project lives.

Develop

This is the branch that all the feature branches push to, it holds a somewhat stable version of the project.

Feature

Feature branches is where most of the work is done by developers and is pushed to develop.

Release

This branch is where the actual release of the project lives, and is forked from develop.

Hotfix

The hotfix branch is where small patches can be worked on, and usually stem directly from the master branch.
